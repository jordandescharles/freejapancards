# FREE JAPAN Cards

Application web pour pratiquer vos connaissances en Hiragana et Katakana.

## Fonctionnalités

- **Responsive** : Optimisée pour mobile, tablette et desktop
- **3 catégories** : Basiques, Avec accents, Combinés pour les Hiraganas et Katakanas
- **Sélection flexible** : Par famille ou caractère individuel

## Développement

L'application utilise :
- HTML5, CSS3, JavaScript vanilla
